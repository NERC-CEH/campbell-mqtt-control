# campbellcontrol python API examples

The primary use case for this package is behind a command-line interface, but it can also be used directly in python scripts.

These examples are offered to show usage - not intended to be re-used as-is (contain hard-coded paths, etc).

Please see the main project README and documentation for the config file format.

## Examples

* [settings.py](settings.py) - show the value of a settings across a whole network and create a CSV summary

`python examples/settings.py`
